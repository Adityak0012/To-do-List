# ✅ To-Do List App

## 📖 About Project
To-Do List App is a simple and responsive task management web application designed to help users organize daily activities efficiently. The app allows users to add, manage, and track tasks through a clean and user-friendly interface.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks easily
- Simple and responsive UI
- Fast and lightweight performance
- Easy task organization

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## ⚙️ How It Works
1. Enter a task in the input field  
2. Add the task to the list  
3. Mark tasks as completed when finished  
4. Delete unnecessary tasks anytime

- Improve task management experience
- Practice JavaScript DOM manipulation
- Build a responsive and interactive web application
- Strengthen frontend development skills
